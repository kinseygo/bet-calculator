【GitHub Pages 上传指南】

1. 登录你的 GitHub 账号。
2. 创建一个新仓库，例如：bet-calculator。
3. 上传以下文件：
   - index.html
   - manifest.json
   - sw.js
   - readme.txt
4. 提交并进入仓库的 Settings -> Pages。
5. 在 “Source” 下选择 “Deploy from a branch” -> “main” -> “/ (root)”。
6. 保存后，稍等几分钟访问：
   https://你的用户名.github.io/bet-calculator/

页面加载后，即可作为 PWA 安装使用。
