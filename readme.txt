【GitHub Pages 上传说明】

仓库: Kinseygo / bet-calculator

1️⃣ 登录 GitHub 并进入仓库 bet-calculator（或新建同名仓库）。
2️⃣ 上传以下文件到仓库根目录:
   - index.html
   - manifest.json
   - sw.js
   - auto_upload.sh
   - readme.txt

3️⃣ 或运行 auto_upload.sh 一键上传:
   在终端输入:
   bash auto_upload.sh

4️⃣ 上传完成后, 打开:
   https://kinseygo.github.io/bet-calculator/

✅ 即可使用网页版投注计算器 v12.3。
